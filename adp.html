SPPD

ADP Accounting System

Project Detail

  • Company/Institution: PT. Gosantha Global, Yogyakarta
  • Work duration: 10 months (Dec 2016 - October 2017)
  • Team size: 8
  • Tools: ExtJS, PHP, PostgreSQL, Fossil

Description

ADP Accounting System is an application that helps numbers division in PT. Aneka Dharma Persada to hold accounting activity, contract, and production activity at once.

ADP has features such as warehouse management, supplies management, purchase management, sales management, transaction monitoring, and accountong reports.

My contributions in this project are:

  1. Contribute on making material supplies and management flow
  2. Design and implement database for material supplies and sales management features
  3. Design mockup for material supplies and sales management features
  4. Contribute on making system documentation
